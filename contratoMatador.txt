Profundizando

Diseño

Crearemos diseños para la imagen general, la disposición del contenido y la funcionalidad de tu sitio web. Este contrato incluye un diseño principal, y la oportunidad de que puedas realizar dos sesiones de revisiones. Si pasadas estas dos revisiones el diseño aun no te conforma, tendrás que pagarnos por todo el trabajo que hayamos producido hasta el momento y podrás tanto cancelar el contrato o continuar confiándonos las siguientes revisiones en base a una renta a tasa diaria definida en nuestra estimación original.

Plantillas y disposición de contenido con HTML5 y CSS

Si el proyecto incluye marcado HTML y plantillas CSS, se desarrollarán usando los estándares HTML5 y CSS2.1 y CSS3 para estilos. El espectro de navegadores web y dispositivos para el acceso a internet cambia regularmente y nuestro aprendizaje es el de pensar hacia adelante, no hacia atrás. Con esto en mente, realizaremos testeos de todo el HTML y CSS en las versiones actuales de los navegadores más importantes para asegurarnos que hemos aprovechado lo mejor de ellos, Para los usuarios con navegadores o dispositivos más viejos, experimentarán un diseño que es apropiado para las capacidades de su software. Para aquellos que usan Microsoft Internet Explorer 6, esto implica un diseño orientado a lo estríctamente tipográfico, no a la disposición del contenido. También testearemos que funcione correctamente en Apple IPAD.

No testearemos en navegadores viejos o abandonados, como Microsoft Internet Explorer 5 o 5.5 para Windows o Mac, versiones anteriores de Apple Safari, Mozilla Firefox u Opera, a no ser que esté especificado. Si necesitas que consideremos estos navegadores viejos, te cobraremos adicionales en una base diaria en base a nuestras estimaciones originales por todo trabajo adicional de diseño, desarrollo y testeo.
